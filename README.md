# Leia Finance UI

React (Vite + TypeScript) implementation of the **Finance Management & Monitoring System v1** UI blueprint.

## Run

```bash
npm install
npm run dev
```

## Notes

- UI is **dense, dashboard-first**, styled with reusable CSS tokens in `src/styles/core.css`.
- **No left borders** and **no translate-based animations** are used.
- Data is mocked in `src/mock/data.ts` to demonstrate screens and drilldowns.
